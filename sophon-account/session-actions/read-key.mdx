<Update label="Retrieve a session status" description="getSessionStatus()">

  Once you have created a session key, you can retrieve its status by calling <Tooltip tip="https://github.com/sophon-org/sophon-account/blob/main/packages/account-core/src/sessionHelper.ts#L54">[getSessionStatus](https://github.com/sophon-org/sophon-account/blob/main/packages/account-core/src/sessionHelper.ts#L54)</Tooltip> function.

  <Info>
    The returned value is an enum of the session key's status, that can be one of the following:

    ```typescript
    export enum SessionStatus {
        NotInitialized = 0,
        Active = 1,
        Closed = 2,
        Expired = 3,
    }
    ```
  </Info>

  ```typescript
  const sessionStatus = await getSessionStatus({
      accountAddress,
      sessionConfig,
      testnet,
  });
  ```
</Update>

<Update label="Retrieve a session state" description="getSessionState()">

  If you are interested in the state of a session key, you can retrieve its state by calling <Tooltip tip="https://github.com/sophon-org/sophon-account/blob/main/packages/account-core/src/sessionHelper.ts#L30">[getSessionState](https://github.com/sophon-org/sophon-account/blob/main/packages/account-core/src/sessionHelper.ts#L30)</Tooltip> function.

  <Info>
    The returned value is an object of the session key's state, that contains the following:

    ```typescript
    export type SessionState = {
        status: SessionStatus;
        feesRemaining: bigint;
        transferValue: {
            remaining: bigint;
            target: Address;
            selector: Hash;
            index: bigint;
        }[];
        callValue: {
            remaining: bigint;
            target: Address;
            selector: Hash;
            index: bigint;
        }[];
        callParams: {
            remaining: bigint;
            target: Address;
            selector: Hash;
            index: bigint;
        }[];
    };
    ```
  </Info>

  ```typescript
  const sessionState = await getSessionState({
      accountAddress,
      sessionConfig,
      testnet,
  });
  ```
</Update>
